Como funciona

  1. Faça upload do documento PDF que precisa ser assinado.
  2. Digite o nome completo do signatário.
  3. Escolha o estilo de assinatura desejado.
  4. O sistema irá encontrar automaticamente os locais para assinatura.
  5. As áreas de assinatura são identificadas por:
    • Linhas com underscores (_____)
    • Linhas com hífens (-----)
    • Textos contendo a palavra "assinatura"
