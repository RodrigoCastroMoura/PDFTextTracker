Detector de Linhas de Assinatura em PDF

Faça upload de um arquivo PDF para encontrar possíveis linhas de assinatura.
Se preenchido, este texto será adicionado acima de cada linha de assinatura encontrada.

Instruções

  1. Faça upload do seu documento PDF usando o seletor de arquivo acima.
  2. Opcionalmente, digite um texto para ser adicionado acima das linhas de assinatura.
  3. O sistema irá procurar por possíveis linhas de assinatura no documento.
  4. As linhas de assinatura serão identificadas com base em:
    • Linhas com underscores (_____)
    • Linhas com hífens (-----)
    • Textos contendo a palavra "assinatura"

Observações Importantes

  • O sistema identifica automaticamente áreas prováveis para assinatura.
  • O aplicativo suporta PDFs normais e documentos digitalizados.
  • Se fornecido, o texto será adicionado 0.5mm acima de cada linha de assinatura.
